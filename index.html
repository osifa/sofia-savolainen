Design and code Sofia Savolainen (2024)

Sing!

Sing! -idea jäi mieleeni Ruotsissa pelaamastani Sing sing-pelistä ( https://www.ninjaprint.se/en/boardgames/sing-sing ), josta kaipasin Suomen kielistä versiota. Kaipasin myös itselleni sovellusprojektia ja päätin tehdä valmistujaisjuhlieni ohjelmanumeroksi kyseisen sovelluksen. Tein hieman sääntömuutoksia, vaikka tässä versiossa (ei ainakaan vielä) ei ole pisteidenlaskua, tärkeintä on pitää hauskaa! Aikaa ei ollut paljoa, joten design hahmottui aika pikaisesti, sekä kuvien piirtäminen.

Peli on koe-pelattu pariin kertaan, ja se jäi vieraiden mieliin.

Peli sisältää tällä hetkellä 150 Suomalaista enemmän ja vähemmän tuttua laulua monelta eri vuosikymmeneltä. Suunnitelmissa on tehdä myös versio englannin kielisistä sekä rock -biiseistä.

Alta löydät esittelyvideon pelistä, tekijänoikeusasioiden takia video on epäselvä ja vain nyt omaan käyttöön, enkä voi koodiakaan laittaa siksi julkiseksi.

Esittelen mielelläni koodit sekä sovelluksen, kun tavataan!

Viikkolukujärjestys

Weekly Schedule is a practical calendar template designed to support busy family life. It was created from a real need and has been used in our everyday life for several years.

Design and code Sofia Savolainen (2024)

Viikkolukujärjestys

Weekly Schedule is a project that has supported our busy family life for several years. Over time, I received multiple requests to make the calendar available for other families as well.

The calendar allows users to mark recurring activities and then download the schedule as a PDF using a button. The PDF layout repeats the schedule three times on a single A4 page, making it easy to print and place on a wall or fridge door. Additional notes can be added later with a pen if needed.

The project is still a work in progress.

Kymppiparit

Kymppiparit is an educational game focused on learning number pairs that sum up to ten. The app was created from a simple need and evolved into a functional mobile game.

Once all pairs are collected, the level is completed and the next level introduces more pairs. The app was built as an Android Angular application. The game is available for testing via Netlify. It is primarily designed for mobile use.
